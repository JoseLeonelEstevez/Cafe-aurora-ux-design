Proyecto desarrollado para el **Laboratorio 1** del programa **Talento Digital**. El objetivo principal fue diseñar una interfaz funcional aplicando principios de usabilidad, accesibilidad y jerarquía visual.

## 🛠️ Principios Aplicados
* **Heurísticas de Nielsen:** Visibilidad del estado del sistema, consistencia y control del usuario.
* **Accesibilidad (WCAG):** Contraste de colores optimizado y tipografía legible (mínimo 16px).
* **Diseño Centrado en el Usuario:** Creación de un menú inclusivo (Pet Friendly) y acceso rápido a conectividad Wi-Fi.

## 📂 Estructura del Proyecto
* **Home:** Landing page con propuesta de valor y acceso a servicios.
* **Menú Premium:** Catálogo de productos y sección especial para mascotas.
* **Info de Red:** Pantalla técnica con credenciales de Wi-Fi Pro.
* **Contacto:** Enlace directo a redes profesionales (LinkedIn).
